05.11.2025

    1.  Система мониторинга телеметрических параметров нефтегазового оборудования на базе Telegram Bot API
    2.  Роли:
        - оператор (получает уведомления о падении того или иного источника телеметрических данных и инструкцию по устранению сбоя);
        - система мониторинга (автоматическая проверка базы данных).
    3.  Функции: оператор
        - регистрация в боте через команду /start;
        - подписка/отписка на уведомления;
        - просмотр текущего состояния телеметрических параметров (когда происходило последнее обновление по каждому источнику);
        - получение автоматических алертов о сбоях на источниках;
        - получение соответствующей инструкции на случай падения каждого из источников.
    Система мониторинга:
        - ежеминутная проверка PostgreSQL (проверка выборки ключевых параметров по каждому источнику);
        - генерация алертов при нахождении сбоев (телеметрический параметр обновлялся позднее, чем 15 минут назад);
        - отправка уведомлений пользователям, подписанным на бот;
        - вывод инструкций по устранению сбоя согласно источнику;
        - логирование событий.
    4.  Базовые функции: фоновый мониторинг времени обновления телеметрических параметров в базе данных;
        автоматическая отправка уведомлений при нахождении хотя бы одного сбоя;
        предоставление статуса параметров по запросу пользователя;
        управление подпиской пользователей на уведомления.
    5.  Пользователь отправляет "/start".
        Бот пишет: "Добро пожаловать в систему мониторинга телеметрии! Введите /help для списка команд."
        Пользователь отправляет: "/status"
        Бот пишет: "Все параметры актуальны. Последняя проверка: 05.11.2025 12:03:40"
        Бот автоматически пишет: "⚠️ АЛЕРТ: Параметр 'Давление на скважине №5' не обновляется уже 15 минут 10 секунд. Последние данные: 05.11.2025 11:48"
        Пользователь отправляет: /unsubscribe
        Бот пишет: "Вы отписаны от уведомлений о сбоях телеметрии"
